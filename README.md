# Knight-Man

A fast-paced coin collection survival game built with Pygame and deployed to the web using pygbag.

## Play Now
https://cellohh.github.io/knight-man/

## Controls
- Arrow Keys / WASD – Move
- Shift + G – God Mode (Cheat)
- R – Restart

## Features
- Boss at 50 coins
- Multiple enemies
- Particle effects
- Cheater vs Legit endings
